# 2026-06-02

## 虚拟时间模型实现
- 完成了可插拔的虚拟时间模型系统
- 新增 `fl_space/fl/time_model.py`：TimeModel(ABC) + SlotTimeModel（方案A+C）+ PhysicsTimeModel（方案B）
- SlotTimeModel：timeslot级粗粒度，配置 slots_per_epoch/slots_per_mb_down/slots_per_mb_up，带时间分解输出
- PhysicsTimeModel：FLOPs/带宽驱动的秒级物理精度，支持硬件参数配置
- TimeBreakdown：单轮时间分解数据结构
- 支持自定义文件导入：`TimeModel.create("path/to/file.py:ClassName")`
- 集成到 FLServer.run_sync、FLConfig、FLRunner、CLI（--time-model, --time-model-args）
- 全部测试通过（4种模式 × 5轮MNIST验证）
